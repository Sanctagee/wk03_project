School Management API

This API manages student records and course listings for GabbyTech Academy. It is built with Node.js, Express, and MongoDB (Mongoose), with data organised across four collections. Week 4 extended the project with Google OAuth 2.0 authentication, so write operations are now protected and only accessible to signed-in users. Sessions are stored in MongoDB and user accounts are created automatically on first login. Use the Swagger UI link to explore and test all endpoints.

Authentication — /auth

GET /auth/google Sign in with Google
GET /auth/profile View current session profile
GET /auth/logout End the current session

Students — /students

GET /students Get all students
GET /students/:id Get a single student
POST /students Add a new student
PUT /students/:id Update a student
DELETE /students/:id Remove a student

Courses — /courses

GET /courses Get all courses
GET /courses/:id Get a single course
POST /courses Add a new course
PUT /courses/:id Update a course
DELETE /courses/:id Remove a course
